# AI Web Insights API

A FastAPI service that scrapes a website homepage and uses AI (GPT-4) to extract key business insights and answer follow-up questions.

## Features
✅ Homepage scraping  
✅ LLM-powered analysis  
✅ Conversational API  
✅ Auth + rate limiting  
✅ Async + FastAPI  

## How to Run
```bash
uvicorn app.main:app --reload
